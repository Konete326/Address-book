Address Book Project Documentation
================================

Project Overview
----------------
A comprehensive address book web application that allows users to store and manage contact information. The project features a user-friendly interface with modern design elements and responsive layout.

Installation Instructions
------------------------
1. Install XAMPP on your system
2. Clone the repository to c:\xampp\htdocs\address-book
3. Start Apache and MySQL services in XAMPP
4. Import the database schema from the SQL file
5. Access the application through localhost/address-book
